Задание 4. "Истязание умножением матриц."
Время работы (8 потоков, матрица 2048*2048):
    матрица хранится по строкам: 375 секунд
    матрица хранится по блокам (128*128 блоков): 88 секунд
Объяснение:
    При умножении в одной из матриц просматриваются элементы, находящиеся в каком-то одном слолбце. При этом процессор кэширует элементы, находящиеся рядом с заданным, так как они попадают в одну строку кэша. Но из-за того, что элементы матрицы хранятся по строкам, в соседних ячейках памяти будут находиться элементы из других столбцов, и они не нужны при вычислении текущей ячейки произведения матриц.
    При хранении по блокам размер каждого блока значительно меньше размера всеё матрицы, и при считывании значения ячейки процессор может закэшировать более одной строки из текущего блока, поэтому следующий в заданном столбце элемент можно считать из кэша, а не из памяти.
